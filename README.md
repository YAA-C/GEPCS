## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/YAA-C/GEPCS.git
~~~

Go to the project directory  

~~~bash  
  cd GEPCS
~~~

Install dependencies  

~~~bash  
./install.bat
~~~

Enable environment  

~~~bash  
./venv/Scripts/activate
~~~
